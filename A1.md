## Backend

### Эндпоинты

| Метод | URL | Описание | Auth |
|-------|-----|----------|------|
| POST | `/api/auth/register/` | Регистрация | Нет |
| POST | `/api/auth/login/` | Вход | Нет |
| POST | `/api/auth/token/refresh/` | Обновить access-токен | Нет |
| GET | `/api/auth/me/` | Данные текущего пользователя | JWT |
| PUT | `/api/auth/me/` | Обновить профиль | JWT |

### Модель User

| Поле | Тип | Описание |
|------|-----|----------|
| `email` | EmailField (unique) | Логин |
| `name` | CharField | Имя |
| `age` | PositiveIntegerField | Возраст (опционально) |
| `city` | CharField | Город (опционально) |
| `photoUrl` | URLField | Ссылка на фото (опционально) |
| `bio` | TextField | О себе (опционально) |

### Запуск

```bash
# 1. Установить зависимости (из корня проекта)
pip install -r requirements.txt

# 2. Перейти в Backend и сделать миграции
cd Backend
python manage.py makemigrations users
python manage.py makemigrations likes
python manage.py migrate

# 3. Запустить сервер
python manage.py runserver 8080
```

## Frontend

### Структура файлов

```
Frontend/src/
  main.ts                              — bootstrap приложения
  index.html
  styles.css
  app/
    app.ts                             — AppComponent (router-outlet)
    app.config.ts                      — провайдеры: router, http, interceptor
    app.routes.ts                      — маршруты /login, /register
    models/
      user.model.ts                    — интерфейс User
    services/
      auth.service.ts                  — login(), register(), logout(), isLoggedIn()
    guards/
      auth.guard.ts                    — authGuard (canActivate)
    interceptors/
      auth.interceptor.ts              — добавляет Bearer токен в каждый запрос
    components/
      login/                           — страница входа
      register/                        — страница регистрации
```

### AuthService — публичное API

```ts
auth.login(email, password)   // → Observable<AuthResponse>
auth.register({ name, email, password, age? })  // → Observable<AuthResponse>
auth.logout()                 // очищает токены из localStorage
auth.isLoggedIn()             // → boolean
auth.getToken()               // → string | null
```

### Как добавить защищённый маршрут (для A2 / A3)

В `app.routes.ts` добавить:

```ts
import { authGuard } from './guards/auth.guard';

{
  path: 'feed',
  loadComponent: () => import('./components/feed/feed').then(m => m.FeedComponent),
  canActivate: [authGuard],
},
```
